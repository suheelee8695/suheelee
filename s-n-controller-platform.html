S.N Controller Platform

This project showcases selected screens from the MXC Supernode Controller platform — a tool designed for developers and DataDash App users to seamlessly connect IoT devices to the internet via the MXProtocol. The platform allows users to provision devices, configure LoRaWAN profiles, manage MQTT integrations, and monitor live device data through a global supernode network.

The following images highlight key features and UI components across different parts of the platform:

Login & Supernode Selection
UI examples for user login and password recovery, including dropdown selections for different regional supernodes (Europe, North America, Asia).

Miner Overview
Shows detailed miner (gateway) information, status, and data logs, which support the communication between IoT devices and the network.

‍

Live Device Activity & Location Map
Includes real-time data visualizations of device communications and geographic coverage using a hex-based map system.

‍

Device Profile & Integration Setup
Displays how users can view device metadata, select device profile settings, and configure MQTT integration with secure credentials.

Provision ID Setup
Screens show the step-by-step flow for creating a Provision ID, including selecting device type, frequency plan, LoRaWAN version, and payload codec.
