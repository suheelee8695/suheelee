Blockchain Launchpad Platform UI

This project showcases the full product design of KILN, a launchpad platform built on the Klaytn blockchain. I designed both the landing experience and application interface, covering the full user journey from onboarding to token staking and trading.

The screens include:

Landing Page & Introduction
A public-facing site introducing the KILN brand, tier system, and tokenomics with soft gradients and playful 3D visuals to appeal to both new and experienced crypto users.

App Dashboard
Core features like token staking, wallet connection, rewards claiming, and liquidity trading are shown in a clean dark UI optimized for web3 users.

IDO Participation Flow
Screens for joining Initial DEX Offerings (IDOs), including upcoming project listings, progress tracking, and token prices.

Error & Utility Pages
Custom designs for 404 and Under Construction pages, ensuring consistent visual branding even in edge cases.
