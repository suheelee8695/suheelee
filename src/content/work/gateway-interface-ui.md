This project presents the desktop application interface for managing MatchX gateways (MXBOX UI), used in LoRa-based IoT networks. The UI was designed to offer clear system visibility and streamlined configuration workflows for network administrators working directly from desktop environments.

The following screens highlight key parts of the interface.

#### Dashboard overview
Real-time status for firmware, network connectivity (Ethernet/Wi-Fi), LoRa packet forwarding, and environmental sensor data: all essential for operational monitoring.

<figure><img src="/images/ezgif.com-gif-maker.gif" alt="Dashboard overview" /></figure>

#### LoRa server configuration
A setup screen for entering server address, ports, and pushing data intervals. Includes an "Auto Config" feature to simplify the onboarding of devices to the backend.

<figure><img src="/images/ezgif.com-gif-maker-1.gif" alt="LoRa server configuration" /></figure>

#### System settings panel
Password management, factory reset, reboot actions, and options to download LoRa/system logs for troubleshooting and secure maintenance.

<figure><img src="/images/ezgif.com-gif-maker-2.gif" alt="System settings panel" /></figure>
